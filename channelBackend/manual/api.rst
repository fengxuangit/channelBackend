API文档
================================

统计安装量
--------------------------------

- POST /api/install

- 请求参数::
	
	
    	{
       	 	"channel": "1"
	    }
    


- 响应结果::

		
    	{
       	 	"status":"success"|"Failed", #返回的状态
       	 	"code":200|100, #返回的状态吗 100为失败
        	"data":0
    	}

- 请求参数::

    	{
